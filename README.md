# Pocket Apps

Personal app store for Mudita Kompakt.

## How it works

1. Install Pocket Apps on your device
2. Connect to WiFi
3. Browse and download apps wirelessly

## Apps

| App | Description | Status |
|-----|-------------|--------|
| Kompakt Games | Game collection | Coming Soon |
| Kompakt Crypto | Encryption tools | Coming Soon |

## Adding apps

1. Upload APK to this repo (root folder)
2. Edit apps.json with app info
3. Commit changes
